# 使用 CSV 管理 i18n 多语言

项目中使用 CSV 文件将多语言集中在一起，方便同时对一个 Key 进行修改。  
通过一个可执行的文件`icotjo`，将 CSV 转换成各语言的 JSON 文件。  
`icotjo`是一个 Go 写的工具，源码可在[https://github.com/wonsikin/icotjo](https://github.com/wonsikin/icotjo)查看。
